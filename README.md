#Data Structures

Some Basic Data Structures for use in JavaScript

##Install
```
npm install data-structures-js
```

###Description
The Library provides the following Data Structure Implementations

1. Stack
2. Queue
    1. Queue
    2. Dequeue
    3. Priority Queue
3. Linked List
    1. Linked List
    2. Doubly Linked List
4. Tree
    1. Binary Search Tree
    2. Red-Black Tree
5. Heap
    1. Max-Heap
    2. Min-Heap

##API

###Stack
* push
* pop
* isEmpty
* size
* toString

###Queue
* enqueue
* dequeue
* isEmpty
* size
* toString

###Dequeue
* insertFront
* insertRear
* removeFront
* removeRear
* isEmpty
* size
* toString
